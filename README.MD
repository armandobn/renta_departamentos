# Pagina de Rentas

### Tegnologias manejadas
* HTML5
* CSS3
* PHP
* JAVASCRIPT
* BOOSTSTRAP 5
* JQUERY
* MARIADB
* SWEETALERT2
* FONTAWESOME
* API GOOGLE MAPS

### Requerimientos
1. Debes tener XAMPP instalado

### Configuraciones basicas
1. Descargar el repositorio
2. Descomprimir el zip
3. Pasar a la carpeta htdocs que se encuentra dentro de Xampp
4. En la carpeta **schema** encontraras un sql que contiene el codigo para crear la BD utilizada
5. Ya puedes utilizar el proyecto

### Funcionamiento
* Mostrar anunciones de Casa/Cuarto que esten sin rentar
* Sistema de Usuarios
* Ubicacion del lugar utilzando la API de Google Maps
* Agregar, Actualizar, Eliminar Anuncios
* Rutas Amigables
* Validaciones de Campos
